Meu Currículo em React

Este é um projeto desenvolvido em React para exibir meu currículo de forma interativa e moderna. O objetivo é apresentar minhas experiências, habilidades e projetos de maneira dinâmica e responsiva.

Tecnologias Utilizadas

React.js - Biblioteca principal para a construção da interface.

Vite - Ferramenta para build e desenvolvimento rápido.

Tailwind CSS - Estilização moderna e responsiva.

React Router - Para navegação entre páginas.

Framer Motion - Animações suaves e interativas.

TypeScript - Para tipagem e segurança no desenvolvimento.

Como Executar o Projeto

Clone este repositório:

git clone https://github.com/seu-usuario/curriculo-react.git

Acesse o diretório do projeto:

cd curriculo-react

Instale as dependências:

npm install

Execute o projeto em modo de desenvolvimento:

npm run dev

Abra o navegador e acesse http://localhost:5173.

Funcionalidades

Exibição de informações pessoais e de contato.

Seção de experiências profissionais e acadêmicas.

Portfólio de projetos realizados.

Layout responsivo e interativo.

Animações para uma melhor experiência do usuário.
